Tilted Plane-parallel Plate
===========================

The tilted plane-parallel plate is characterized by its thick `L`, index of refraction `n`, and beam falling angle `Î±`. The element is similar to :doc:`ElemBrewsterPlate`, but the falling angle is not fixed and can be set via the element parameter.
